# Ejercicio Guillermo Sequeira

> Este proyecto muestra y administras las distintas subscripciones a partners que puede adquirir el usuario, también muestra los descuentos que puede tener según su nivel
## Development

### 1) Build the assets:

```
npm run watch
```

### 2) Run the app:

```
npm run start-dev
```

### 3) Navigate to:
```
http://localhost:3000/
```
### 4) Testing
```
npm run test
```
